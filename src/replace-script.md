# Text Color Replacement Map

This document tracks the replacement of dark text colors with white throughout the platform.

## Replacement Rules:
- `text-slate-300` → `text-white/90`
- `text-slate-400` → `text-white/80`
- `text-slate-500` → `text-white/70`
- `text-slate-600` → `text-white/60`
- `text-slate-700` → `text-white/50`
- `text-gray-300` → `text-white/90`
- `text-gray-400` → `text-white/80`
- `text-gray-500` → `text-white/70`
- `text-gray-600` → `text-white/60`
- `text-gray-700` → `text-white/50`
- `text-black` → `text-white`
